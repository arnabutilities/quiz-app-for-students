# Syllabus & Question Formulation Guide
## Young Internet Explorers Curriculum

This guide defines the educational design parameters for formulating, verifying, and logging exam questions in the **Young Internet Explorers** Class 8 curriculum list.

---

## 📖 Class 8 Curriculum Mapping: "Internet and Its Use"
The default questions address four critical learning areas based on national digital curriculum specs for middle schoolers:

1. **Web Fundamentals & Protocols**: Identifying the parts of an address (WWW, DNS, HTTP, HTTPS).
2. **Network Hardware**: Understanding how information journeys across routers and IP addresses.
3. **Cyber Hygiene & Safety**: Recognizing email spam, phishy links, and browser cookie functions.
4. **Protective Mechanisms**: Learning how firewalls keep networks and individual machines safe.

---

## 🙋 Designing Effective Syllabus Questions
When a teacher adds a custom question via the **Syllabus Editor Desk**, they are required to submit five fields:

### 1. Question Prompt Text
- **Goal**: Clear, contextual scenarios that encourage logical deduction.
- **Example**: *"You receive an email from a stranger stating you inherited $1,000,000. What is this message likely considered?"*

### 2. Multi-Choice Options (A, B, C, D)
- **Goal**: One clear correct choice, coupled with three realistic distractors that address common student misconceptions.
- **Example**:
  - `A`: A genuine prize notification from Google's CEO.
  - `B`: An email spam / phishing attempt designed to steal personal info. *(Correct)*
  - `C`: A certified homework assignment notification.
  - `D`: An automatic antivirus guard alert.

### 3. Correct Choice Selector
- **Goal**: A dropdown matching the correct letter ('A', 'B', 'C', or 'D').

### 4. Interactive Solution Explanation Info
- **Goal**: Immediate, positive coaching feedback displaying real-world computer science rules. Exposing *why* the correct answer is secure.
- **Example**: *"Never click links on stranger emails. Spam messages often pretend to be wealthy giveaways to trick kids into sharing passwords."*

### 5. Media Context Tags (`graphicKeyword` & `helpInstruction`)
- **Keywords determined automatically**: Let the system render an interactive visual graphic based on the keyword category ('web', 'safety', 'network', 'email', 'search', 'hardware') to help young visual learners link text prompts with schematic concepts.
